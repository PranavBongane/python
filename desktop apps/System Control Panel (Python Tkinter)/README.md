# ⚡ System Control Panel (Python Tkinter)

A modern desktop utility built with Python and Tkinter that allows users to **Restart, Shutdown, Log Out, and schedule Restart** with a clean and user-friendly GUI.

---

## 🚀 Features

* 🔄 **Instant Restart**
* ⏳ **Restart with Delay (20 seconds)**
* 🚪 **Log Out**
* ⛔ **Shutdown System**
* ⚠️ **Confirmation Dialogs** before every action
* 🎨 **Modern Dark UI** with styled buttons
* 🖱️ Simple and interactive interface

---

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (GUI framework)
* OS module (system commands)

## ▶️ How to Run

1. Clone the repository or download the file

```
git clone https://github.com/your-username/system-control-panel.git
cd system-control-panel
```

2. Run the script

```
python main.py
```

---

## ⚠️ Important Notes

* This application uses **system-level commands**, so:

  * It works primarily on **Windows OS**
  * May require **administrator permissions**
* Be careful while testing (it can restart/shutdown your system)

---

## 🎨 UI Preview

* Dark theme interface
* Color-coded buttons:

  * Green → Restart
  * Blue → Timed Restart
  * Orange → Log Out
  * Red → Shutdown

---

## 🔐 Safety Features

* Confirmation popup before executing any action
* Prevents accidental shutdown/restart
